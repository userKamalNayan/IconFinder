# IconFinder
APK Link -> https://drive.google.com/file/d/1i5HlwpavFxcPmcT9Tk7U9SxHAL0-aD-Q/view?usp=sharing

A demo app in **Kotlin** using **Modular** and **Clean Architecture** with **Use Case Pattern** to demonstrate use of **IconFinder** api using **retrofit** with **pagination**.
In the app some libraries are being used some of them are:
**Koin** is used for Dependency Injection, **Timber** for debugging purposes, **Epoxy RecyclerView** to show data.
